## ðŸŒŸ What is Reddit Wrapped?

Reddit Wrapped is an interactive web experience that analyzes your Reddit activity and presents it in a beautiful, engaging slideshow format. Get insights into your Reddit journey with comprehensive statistics, personality analysis, and achievement badges - all wrapped up in a stunning visual experience!

### Tips for Best Results

- Works with any **public** Reddit profile
- Fetches all available posts and comments (Reddit API may limit to ~1000 most recent)
- Results are cached for 1 hour for faster loading
- Try well-known accounts like `spez`, `reddit`, or your own!
